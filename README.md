# simple webpack react babel webserver hmr proxy
## Features

* React 16
* Webpack 4
* Babel 7
* Hot Module Replacement
* Proxy
* css

## Installation

* `git clone git@github.com:alexej3ajtsev/simple-react-boilerplate.git your_folder`
* cd your_folder
* npm install
* npm run start
* visit `http://localhost:8080/`
